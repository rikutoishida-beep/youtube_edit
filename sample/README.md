# SRT字幕校正サンプル

このフォルダは、AI文字起こし字幕の校正・最適化の見本セットです。

## 📁 ファイル構成

```
sample/
├── input/
│   └── Bパターン_4.srt          ← AI文字起こしの元ファイル
├── output/
│   ├── 話者1（ごますけ）.srt     ← 校正後：話者1用
│   └── 話者2（白臨こと）.srt     ← 校正後：話者2用
├── README.md                   ← このファイル
└── ../process_srt.py           ← 処理スクリプト（親フォルダ）
```

## 📋 処理内容

このサンプルでは以下の処理を実施しています：

### 1. 固有名詞・名前の修正
- `シリンさん` / `しりんさん` → `白臨さん`
- `ゴマスケさん` → `ごますけさん`
- `こたさん` → `ことさん`
- `まことさん` → `ことさん`

### 2. ケバ取り・フィラー除去
- 「あの、」「えー」などの無意味なフィラーを削除
- 「守秘義務遵守の、守秘義務遵守のため、」のような言い直しを統合
- 「職場でストレス--職場でストレス」のような重複を修正

### 3. テキスト最適化
- 断片化されたテキストを自然な文単位に結合
- 1文15～20文字の目安で読みやすく分割
- 「世の中」などの複合語を分割しない

### 4. 話者別分離
- 元のSRTから話者タグを除去
- 各話者のセリフのみを抽出して別ファイルに出力
- 連番を1から振り直し

### 5. タイムコード配分
- 元ブロックのタイムコード情報を保持
- 文字数と話速比率でタイムコードを按分
- 最小1秒の表示時間を保証

## 🔧 使用方法

`process_srt.py` を使って同じ処理を再現できます：

```bash
python3 process_srt.py
```

処理の詳細は `claude.md` を参照してください。
